首先，要了解我们所谓的网页到底是由那几个部分组成

前言

日常生活中接触到的网站中能看到的部分是由HTML，Css和Js组成的，也称之为前端页面。

  • Html:HTML(HyperText Markup Language)的缩写，它来决定我们网页中的结构。
  • Css:CSS(Cascading Style Sheets)的缩写,它是决定Html是如何显示的。
  • Js:Js(Javascript)的缩写,它是网页开发中的核心技术也是一门开发语言。

Html是用来描述人体器官的。
Css是用来确定各器官的位置、形状、大小及颜色的。
Js是用来决定哪些器官可以有哪些行为动作和可以产生怎样的交互的。

让我们了解一下Html和Css的具体语法

基础语法

Html中大部分标签都是用<标签名></标签名>的形式来表达。

<html></html>  <div></div>  <p></p>

Css的语法，今天先介绍一部分哈。

我们说过Css是描述结构的样式的，简单点可以理解为化妆。它可以为<眼睛>、id=“左眼”和class=“眼睛”等添加不同的样式。 

//下面是描述<eye></eye>的样式
eye:{ 
color: black;
width: 20px;
heigth:30px;
}
//以#+id的形式描述具体某一个标签<eye id="左眼"></eye>的样式
#左眼{
  color:yellow;
}
//以.+class名的形式描述具体某一类标签<eye class="眼睛"></eye>的样式
.眼睛{
  color:red;
}
